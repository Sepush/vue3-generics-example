# Vue3 generics

demos to show ways to use generics in Vue3 with TypeScript

1. Using generics in script setup
2. Using generics in defineComponent
3. Using generics to wrapped TSX component

### Setup

```bash
corepack prepare
pnpm i
pnpm run dev
```
